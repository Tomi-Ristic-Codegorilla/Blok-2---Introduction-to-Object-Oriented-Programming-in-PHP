Bijlage 4 - Exercises Hier een aantal oefeningen zodat je in de praktijk een aantal praktische classes kan schrijven. Basic Class Exercises De onderstaande opdrachten zijn bedoeld om kennis te maken met het schrijven van OOP code. Opdracht 1 Write a simple PHP class which automatically displays the following string when initialized. Output: 'MyClass class has been initialized!' Opdracht 2 Write a simple PHP class which displays an introductory message like "Hello All, I am Scott", where "Scott" is an argument value of the method within the class. Opdracht 3 Write a PHP calculator class which will accept two values as arguments, then add them, subtract them, multiply them together, or divide them on request. For example : $mycalc = new MyCalculator( 12, 6); echo $mycalc- > add(); // Displays 18 echo $mycalc- > multiply(); // Displays 72
